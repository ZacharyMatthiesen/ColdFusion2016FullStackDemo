Exercise 6.0

 

Objectives: To become familiar with the cfmail and cfhttp tags.

 

 

Requirements

 

1.       Read first names from the database making sure that one of the names is “The” before you start the exercise.

2.       Read www.cnn.com into a variable.

3.       Replace all words in the variable that match any of the first names with “MOD”.

4.       Output the variable to the screen.

5.       Email the variable contents to yourself.

 

Implementation

 

1.       Query your database.

2.       Get your URL.

3.       Find and replace all instances of the first names on the web page.

Tip: Exercises 4 & 5 may help with this.

4.       Show the results on the screen.

5.       Email the results to yourself.
